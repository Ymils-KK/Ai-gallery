# 设计规范

## 色彩方案

### 主色
| 用途 | 色值 | 说明 |
|------|------|------|
| 背景色 | `#0a0a0f` | 深色主背景 |
| 卡片背景 | `#14141f` / `rgba(255,255,255,0.03)` | 半透明卡片 |
| 文字主色 | `#e4e4e7` | 主要文字 |
| 文字次色 | `#a1a1aa` | 次要文字/日期 |

### 霓虹点缀
| 用途 | 色值 | Tailwind |
|------|------|----------|
| 紫色主调 | `#a855f7` | purple-500 |
| 蓝色辅调 | `#3b82f6` | blue-500 |
| 青色辅调 | `#06b6d4` | cyan-500 |
| 渐变主色 | `from-purple-500 via-blue-500 to-cyan-500` | 标题/按钮渐变 |

### 边框
| 用途 | 色值 |
|------|------|
| 卡片边框 | `rgba(255,255,255,0.06)` |
| 悬停边框 | `rgba(168,85,247,0.3)` (紫色辉光) |

## 字体

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| H1 | `text-4xl md:text-5xl lg:text-6xl` | bold (700) | 页面大标题 |
| H2 | `text-2xl md:text-3xl` | bold (700) | 区块标题 |
| H3 | `text-xl md:text-2xl` | semibold (600) | 卡片标题 |
| body | `text-base` | normal (400) | 正文 |
| small | `text-sm` | normal (400) | 次要信息 |

字体族：系统默认无衬线（Geist 作为主字体）

## 动效规范

| 场景 | 效果 | 时长 |
|------|------|------|
| 卡片悬停 | scale(1.02) + 边框辉光 + 阴影 | 300ms ease-out |
| 页面加载 | 从下淡入 (translateY + opacity) | 500ms |
| 灯箱打开 | 缩放淡入 | 300ms |
| 筛选切换 | 背景色过渡 | 200ms |
| 导航悬停 | 文字变色 | 150ms |

统一使用 `transition-all duration-300` 作为基础过渡。

## 组件风格

### 导航栏
- 固定在顶部
- 背景：半透明毛玻璃 `backdrop-blur-xl bg-background/80`
- 高度：`h-16` (64px)
- 底部细线分隔

### 卡片
- 圆角：`rounded-xl` 或 `rounded-2xl`
- 边框：`border border-white/[0.06]`
- 悬停：边框变紫 + 微放大 + 阴影

### 按钮
- 主按钮：紫色渐变背景，白色文字
- 次按钮：透明背景，边框，悬停变色
- 圆角：`rounded-lg`

### 标签(Tag)
- 小圆角 `rounded-full`
- 半透明背景 + 边框
- 悬停时背景变亮

## 响应式断点
遵循 Tailwind 默认：
- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px

移动端：单列布局；平板：2列；桌面：3列画廊
