# 开发需求文档

## 项目简介
一个展示AI生成作品（图片、视频）的个人网站，面向零基础用户，采用暗色炫酷风格。

## 目标用户
- 网站所有者：不懂代码，会用 Markdown 写文章
- 访问者：对AI创作感兴趣的人

## 功能需求

### P0 — 必须有
1. **作品画廊（首页）**
   - 网格布局展示 AI 生成的图片和视频
   - 分类筛选：全部 / 图片 / 视频
   - 点击作品打开灯箱查看大图或播放视频
   - 支持占位内容，方便后续替换

2. **关于我页面**
   - 头像 + 自我介绍
   - AI 工具标签（可自定义）
   - 社交链接（GitHub、微博、B站等）

3. **博客系统**
   - 文章列表页，按时间倒序
   - 文章详情页，Markdown 渲染
   - 支持图片、代码块、表格

4. **全局**
   - 顶部导航栏（首页 / 关于 / 博客）
   - 页脚版权信息
   - 移动端响应式

### P1 — 加分项
- 页面切换过渡动画
- 滚动渐入效果
- SEO 基础配置（标题、描述）

## 页面结构
```
首页 (/)          → 作品画廊
关于我 (/about)    → 个人介绍
博客 (/blog)       → 文章列表
博客详情 (/blog/[slug]) → 单篇文章
```

## 非功能需求
- 代码结构清晰，配置集中管理
- 支持后续替换真实内容
- 构建无报错
- 移动端适配良好
